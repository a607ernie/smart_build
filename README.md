# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1xb-HCQPptb3J7cDN9bXC2BsMU9DavlS2

## Run Locally

**Prerequisites:** Node.js (>= 18, recommended 20)

This repo includes a `.nvmrc` (Node 20). If you use `nvm`, run `nvm use` in the project folder.


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
